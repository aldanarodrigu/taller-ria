Requisitos

    Node.js v24
    npm

Instalación

    1. Clonar repo:
       git clone 
       cd 

    2. Instalar dependencias:
       npm install

Variables de entorno

    Crear archivo .env en la raíz con la clave de RAWG: VITE_RAWG_KEY=e6c1c1e93265476a94299c2196710783


Ejemplo (.env.example)
VITE_RAWG_KEY=e6c1c1e93265476a94299c2196710783

Comandos principales

    npm run dev — iniciar servidor de desarrollo (recomendado)
    npm run build — generar build de producción (opcional)
    npm run preview — (opcional) previsualizar build de producción



Dependencias relevantes

    Vue 3, TypeScript, Vite, vue-router, pinia
    Otras: @heroicons/vue, @dicebear, @tsparticles, chart.js, vue-chartjs, @vueuse/motion
